What is the difference between Community and Professional Pro?
The Community Tier provides 3 passive scans daily focusing on HTTP headers, SSL certificates, and basic layout checks. Pro unlocks unlimited deep active scanning, including SQLi matrices, XSS checks, port scanning, and actionable remediation snippets.
How do the Remediation Blueprints work?
When Pro users encounter a vulnerability, Sentinel provides fully copy-pasteable Nginx, Apache, and application-layer code blocks tailored to patch the exact security flaw dynamically.
Is it safe to run Sentinel against my live production servers?
Yes. Sentinel's scans are carefully metered to avoid denial-of-service conditions. However, active injection payloads (like SQLi testing) may safely trigger your web application firewalls (WAF) or generate standard error logs on your backend.
